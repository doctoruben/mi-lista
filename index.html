Conectar a Firebase

Introduce los datos de tu proyecto Firebase para sincronizar la lista entre dispositivos.

1 Ve a console.firebase.google.com
2 Crea un proyecto nuevo (nombre libre)
3 En el proyecto → Configuración ⚙️ → Tus apps → añade una app web </>
4 Copia apiKey, projectId y appId de la configuración
5 Ve a Realtime Database → Crear base de datos → modo prueba
6 Pega los datos aquí ↑ y guarda
